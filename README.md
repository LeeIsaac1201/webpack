# Bootstrap + Webpack Starter

A lightweight, modern frontend setup using **Webpack 5** and **Bootstrap 5**.  
Includes a fast dev server with hot reload, a production build pipeline, and recommended repo practices (linting/formatting ready to enable).

---

## Start

```bash
# 1) Install dependencies
npm install

# 2) Start the dev server (http://localhost:8080)
npm run dev

# 3) Build for production (outputs to /dist)
npm run build
```
